我来给你做一个**可以实际用的Excel模板**，包含自动化计分、可视化和每周更新流程。好的，这是一个可以帮助你实现付鹏宏观流动性过滤的Excel模板。

### 设计理念

1.  **直观性**：一眼就能看出当前宏观流动性的总体态势。
2.  **易用性**：只需填写少量数据，自动计算评分。
3.  **可维护性**：方便你调整评判标准和历史数据。
4.  **动态性**：通过颜色标记，直观反映趋势变化。

### Excel 模板结构

我将为你描述如何在 Excel 中构建这个模板。你可以直接复制这些说明到 Excel 中操作。

---

**工作表名称：** `宏观流动性评分表`

### **A. 数据输入区（每周更新）**

| 单元格 | 内容                                        | 说明                                                              | 数据类型  |
| :----- | :------------------------------------------ | :---------------------------------------------------------------- | :-------- |
| **A1** | **宏观流动性评分表**                        |                                                                   |           |
| **A3** | **指标名称**                                |                                                                   |           |
| **B3** | **最新值**                                  | 本周/本月最新数据                                                 | 数字/百分比 |
| **C3** | **前值**                                    | 上周/上月数据，用于比较趋势                                       | 数字/百分比 |
| **D3** | **趋势**                                    | 自动计算：上涨/下跌/持平                                          | 文本      |
| **E3** | **评分标准（偏多）**                        | 你设定的偏多条件（如：同比>8.5%）                                 | 文本      |
| **F3** | **我的评分（0-2）**                         | 自动计算：0=偏空，1=中性，2=偏多                                  | 数字      |
| **G3** | **备注**                                    | 记录数据来源、参考网站或个人解读                                  | 文本      |
| **A4** | **1. 社融同比 (%)**                         | 央行每月公布。通常以最新月度数据为准。                            | 数字      |
| **A5** | **2. M2-M1剪刀差 (%)**                      | 央行每月公布。                                                    | 数字      |
| **A6** | **3. 10年期国债收益率 (%)**                 | 可每日获取，取周末收盘数据。                                      | 数字      |
| **A7** | **4. 北向资金4周累计净流入 (亿元)**         | 每周更新，查看近4周累计数据。                                     | 数字      |
| **A8** | **5. 两融余额周度变化 (亿元)**              | 每周更新，查看本周与上周两融余额的差值。                          | 数字      |
| **A9** | **6. 美元指数周度变化**                     | 美元指数本周变化（如：上涨1 = 1%）                                | 数字      |
| **A10** | **7. 美国10年期国债收益率周度变化 (%)** | 美债收益率本周变化（如：上涨0.1 = 0.1%）                          | 数字      |

### **B. 自动计算公式**

**D 列 (趋势)：**

*   **D4** (社融同比): `=IF(B4>C4,"上涨",IF(B4<C4,"下跌","持平"))`
*   **D5** (M2-M1剪刀差): `=IF(B5<C5,"收窄",IF(B5>C5,"扩大","持平"))` (收窄为利好)
*   **D6** (10年国债收益率): `=IF(B6<C6,"下跌",IF(B6>C6,"上涨","持平"))` (下跌为利好)
*   **D7** (北向资金): `=IF(B7>C7,"上涨",IF(B7<C7,"下跌","持平"))`
*   **D8** (两融余额): `=IF(B8>C8,"上涨",IF(B8<C8,"下跌","持平"))`
*   **D9** (美元指数): `=IF(B9<C9,"下跌",IF(B9>C9,"上涨","持平"))` (下跌为利好)
*   **D10** (美国10年期国债收益率): `=IF(B10<C10,"下跌",IF(B10>C10,"上涨","持平"))` (下跌为利好)

**F 列 (我的评分)：**

你需要根据自己的经验和付鹏的理念，**设定E列的“偏多”标准**。以下是一些**示例**，你需要根据市场实际情况和个人理解进行调整。

*   **F4** (社融同比): `=IF(B4>8.5,2,IF(B4>=7.5,1,0))` (示例：高于8.5分2，7.5-8.5分1，低于7.5分0)
    *   **E4 填写示例**：`>8.5%`
*   **F5** (M2-M1剪刀差): `=IF(D5="收窄",2,IF(AND(D5="持平",B5<=-1),1,0))` (示例：收窄分2，持平且低于-1%分1，否则分0。M1-M2一般是负的，越负越少越好)
    *   **E5 填写示例**：`收窄或持平且<-1%`
*   **F6** (10年国债收益率): `=IF(D6="下跌",2,IF(AND(D6="持平",B6<3),1,0))` (示例：下跌分2，持平且低于3%分1，否则分0)
    *   **E6 填写示例**：`下跌或持平且<3%`
*   **F7** (北向资金4周累计): `=IF(B7>100,2,IF(B7>0,1,0))` (示例：净流入>100亿分2，净流入>0分1，否则分0)
    *   **E7 填写示例**：`>100亿`
*   **F8** (两融余额周度变化): `=IF(D8="上涨",2,IF(B8>0,1,0))` (示例：上涨分2，净增>0分1，否则分0)
    *   **E8 填写示例**：`上涨或净增>0`
*   **F9** (美元指数): `=IF(D9="下跌",2,IF(B9<0,1,0))` (示例：下跌分2，净跌分1，否则分0)
    *   **E9 填写示例**：`下跌或净跌`
*   **F10** (美国10年期国债收益率): `=IF(D10="下跌",2,IF(B10<0,1,0))` (示例：下跌分2，净跌分1，否则分0)
    *   **E10 填写示例**：`下跌或净跌`

### **C. 汇总与决策区**

| 单元格 | 内容                                        | 说明                                           | 数据类型 |
| :----- | :------------------------------------------ | :--------------------------------------------- | :------- |
| **A12** | **总分**                                    | 自动计算所有指标评分之和                       | 数字     |
| **B12** | `=SUM(F4:F10)`                               |                                                |          |
| **A13** | **评分区间**                                |                                                |          |
| **B13** | **宏观流动性判断**                          | 自动显示判断结果                               | 文本     |
| **B14** | **建议仓位**                                | 自动显示建议仓位                               | 文本     |
| **A15** | **你的最大仓位上限 (%)**                    | **手动输入**，作为实际操作的上限                 | 数字     |
| **A16** | **当前实际仓位 (%)**                        | **手动输入**，用于对比分析                       | 数字     |

**B13 (宏观流动性判断)：**
你可以根据总分设置你的判断逻辑。例如：
`=IF(B12>=10,"积极做多",IF(B12>=6,"谨慎做多",IF(B12>=3,"防守为主","空仓/极度防守")))`
(示例：总分10-14分积极做多，6-9分谨慎做多，3-5分防守为主，0-2分空仓/极度防守)

**B14 (建议仓位)：**
根据判断结果给出建议仓位。你可以参考你的**A15**设定的最大仓位上限来调整。
`=IF(B13="积极做多",TEXT(A15*0.8,"0%"),IF(B13="谨慎做多",TEXT(A15*0.5,"0%"),IF(B13="防守为主",TEXT(A15*0.2,"0%"),TEXT(0,"0%"))))`
(示例：积极做多=最大仓位*0.8，谨慎做多=最大仓位*0.5，防守为主=最大仓位*0.2，空仓=0)

### D. 美化与条件格式

1.  **颜色编码D列（趋势）**：
    *   选中D4:D10区域。
    *   条件格式 -> 突出显示单元格规则 -> 文本包含...
        *   "上涨" -> 绿色填充，深绿字体 (利好)
        *   "下跌" -> 红色填充，深红字体 (利空)
        *   "收窄" -> 绿色填充，深绿字体 (利好)
        *   "扩大" -> 红色填充，深红字体 (利空)
        *   "持平" -> 黄色填充，深黄字体 (中性)
        *   注意：对于国债收益率和美元指数，上涨是利空，下跌是利好，需要反向设置颜色。你可以根据具体情况调整规则。
2.  **颜色编码F列（我的评分）**：
    *   选中F4:F10区域。
    *   条件格式 -> 突出显示单元格规则 -> 等于...
        *   "2" -> 绿色填充，深绿字体
        *   "1" -> 黄色填充，深黄字体
        *   "0" -> 红色填充，深红字体
3.  **颜色编码B13 (宏观流动性判断)**：
    *   选中B13单元格。
    *   条件格式 -> 突出显示单元格规则 -> 文本包含...
        *   "积极做多" -> 绿色填充
        *   "谨慎做多" -> 浅绿色填充
        *   "防守为主" -> 黄色填充
        *   "空仓/极度防守" -> 红色填充
4.  **边框和字体**：增加边框，调整字体大小和加粗，让表格更易读。

### E. 数据获取建议

1.  **社融同比、M2-M1剪刀差**：
    *   **来源**：中国人民银行官网 (www.pbc.gov.cn) -> 统计数据 -> 金融统计数据报告。通常每月发布。
    *   **频率**：每月更新，你可以在公布后第一时间更新到Excel。
2.  **10年期国债收益率**：
    *   **来源**：同花顺、东方财富、Wind等财经软件的行情数据。
    *   **频率**：每周日晚更新上周五收盘数据。
3.  **北向资金4周累计净流入**：
    *   **来源**：东方财富、同花顺、腾讯证券等财经网站，搜索“北向资金”。
    *   **频率**：每周日晚更新。
4.  **两融余额周度变化**：
    *   **来源**：东方财富、同花顺等财经网站，搜索“两融余额”。
    *   **频率**：每周日晚更新。
5.  **美元指数周度变化**：
    *   **来源**：Investing.com、FXStreet等外汇网站。
    *   **频率**：每周日晚更新上周五收盘数据。
6.  **美国10年期国债收益率周度变化**：
    *   **来源**：Investing.com、英为财情等全球金融数据网站。
    *   **频率**：每周日晚更新上周五收盘数据。

### F. 如何使用

1.  **每周固定时间**：选择一个固定的时间（例如每周日晚上），打开你的Excel文件。
2.  **更新数据**：从相应的网站或软件获取最新的数据，填入B列的"最新值"和C列的"前值"（只需填写本周数据，然后把上周的最新值移到前值列）。
3.  **查看结果**：D列、F列、B12、B13、B14会自动更新。
4.  **调整仓位**：根据B14的建议仓位，结合你的实际情况和风险偏好，调整你的A股总仓位。
5.  **记录**：可以在G列或新建一个历史记录工作表，记录每周的打分和你的仓位调整。

这个模板提供了一个起点，你可以根据自己的理解和实践，调整指标、评分标准和仓位建议。祝你交易顺利！
