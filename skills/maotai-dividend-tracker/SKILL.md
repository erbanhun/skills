# 茅台红利持有逻辑周度追踪器

## 触发条件

用户提到以下任一关键词或意图时，使用本 skill：
- 茅台 / 贵州茅台 / 600519 的股息、分红、红利、持有逻辑
- 按周收集茅台数据、周度追踪、定期评估茅台
- 茅台持有评分、红利得分、是否该持有/加仓/减仓
- 茅台浮亏处理、茅台定投、茅台分红健康度检查
- 要求生成茅台红利投资报告、茅台估值评估

## 核心工作流

### 1. 运行追踪脚本

脚本位置：`scripts/maotai_tracker.py`

在终端执行：
```bash
python scripts/maotai_tracker.py --output-dir <输出目录> --json
```

参数说明：
- `--output-dir`: 报告输出目录，默认当前目录
- `--json`: 同时输出原始评分 JSON，方便做时间序列对比

**环境要求**：`pip install akshare pandas numpy`

### 2. 读取并解读报告

脚本会生成两个文件：
- `maotai_dividend_report_YYYYMMDD.md` —— 人类可读的周度评估报告
- `maotai_dividend_data_YYYYMMDD.json` —— 原始评分数据

将 Markdown 报告的核心结论呈现给用户，包括：
1. **总得分**（满分 100）
2. **四大维度拆解**：股息率定价、分红健康度、估值安全垫、趋势技术面
3. **持有建议**：根据得分区间给出的动作指引
4. **决策树自检**：结合当前指标，给出浮亏处理的参考判断

### 3. 评分框架详解（向用户解释时引用）

本评分严格基于《股息类股票的持有与浮亏处理——实战手册》的实战框架：

| 维度 | 权重 | 核心指标 | 得分逻辑 |
|------|------|---------|---------|
| **股息率定价** | 25分 | TTM 股息率 | <1.5%→5分, 1.5-3%→15分, 3-4%→22分, ≥4%→25分 |
| **分红健康度** | 40分 | 净利润增长、经营现金流/净利润、资产负债率、分红率 | 各10分，红灯/黄灯/绿灯分档 |
| **估值安全垫** | 20分 | PE 历史分位、价格历史分位（5年） | <30%→10分, 30-50%→7分, 50-70%→4分, >70%→1分 |
| **趋势技术面** | 15分 | 52周高点比、20周均线趋势 | 位置8分+趋势7分 |

**总分解读**：
- **≥80 分**：🟢 强烈持有 / 积极加仓。时间站在你这边，分红再投资。
- **60-79 分**：🟡 持有观察。核心逻辑未变，某项指标黄灯，暂停新增仓。
- **40-59 分**：🟠 谨慎观察。强制复盘买入逻辑，逻辑已变则重新定仓。
- **<40 分**：🔴 警惕 / 考虑减仓。默认减仓一半，再研究。

### 4. 数据口径说明（回答用户疑问时参考）

- **行情数据**：新浪财经前复权日线，回溯 5 年
- **财务数据**：巨潮资讯季度财务指标，回溯 5 年
- **分红数据**：巨潮资讯历史分红，TTM 股息率 = 最近 12 个月现金分红总额 / 最新收盘价
- **分红率**：年度每股现金分红 / 年度每股收益（取最近年报）
- **PE 近似分位**：基于历史价格与季度每股收益构建的近似 PE 序列，计算 5 年历史分位
- **净利润增长率**：最新季度同比（接口原始值）
- **经营现金流/净利润**：接口列名含 `%`，但部分数据为小数比例（如 0.72），脚本会自动识别并转为 72%

## 故障处理

| 场景 | 处理方式 |
|------|---------|
| akshare 未安装 | 提示用户运行 `pip install akshare pandas numpy` |
| 网络/代理导致东方财富接口失败 | 脚本已优先使用新浪行情 + 巨潮财务，如遇失败可建议用户切换网络重试 |
| 分红数据缺失 | 脚本会降级处理，股息率显示为 0%，并在报告中提示 |
| 财务数据滞后 | 使用最新可用季度，报告中注明「最新季度」日期 |
| 用户想对比历史得分 | 使用 `--json` 输出，建议用户将历次 JSON 保存在同一目录，可用脚本做时间序列可视化 |

## 扩展建议（主动向用户提出）

如果用户表现出持续跟踪的意图，建议：
1. **定时执行**：用 Windows 任务计划程序 / cron 每周一运行一次脚本，输出到固定目录
2. **历史对比**：保存每次 JSON，用 Excel/Python 做得分趋势图，观察「分红健康度」是否持续恶化
3. **组合扩展**：本脚本框架可复用到其他红利股（长江电力、中国神华等），只需修改脚本顶部的 `SYMBOL` 常量
