# A股板块主线追踪器

## 触发条件

用户提到以下任一关键词或意图时，使用本 skill：
- 板块主线 / 市场主线 / 领涨板块 / 最强板块
- 行业轮动 / 主线切换 / 谁是龙头
- 主线持续性 / 主线见顶 / 主线还在吗

## 前置要求

### 1. 指南针活跃市值数据（必需）

本 skill 判断多头区间的**核心依据**是「指南针活跃市值」。用户需在调用时提供 CSV 文件，格式如下：

```csv
date,active_market_cap
2026-04-20,15000.5
2026-04-21,15200.3
2026-04-22,15600.8
2026-04-23,15850.2
2026-04-24,16500.0
```

> 只需提供**最近 30 个交易日**的数据即可。日期格式支持 `YYYY-MM-DD` 或 `YYYYMMDD`。

### 2. 多头区间判定规则（用户指定）

| 状态 | 条件 |
|------|------|
| **开启多头** | 指南针活跃市值单日涨幅 ≥ 4%，或连续 2 日累计涨幅 ≥ 4% |
| **维持多头** | 已开启后，单日跌幅 **不触及 -2.3%**（即 > -2.3%） |
| **结束多头** | 单日跌幅 ≤ -2.3% |

> 辅助参考：上证/沪深300 是否在 20 日均线上方（仅输出供参考，不决定多头状态）。

## 核心工作流

### Step 1: 大盘环境判断

读取用户提供的指南针活跃市值 CSV，计算涨跌幅，按上述规则判定：
- 当前是否处于多头区间？
- 多头持续天数？
- 多头强度（强/温和，基于活跃市值斜率）

**若非多头区间，直接输出观望报告，不进入板块扫描。**

### Step 2: 板块初筛

**行业板块**（约90个）：全部扫描  
**概念板块**（约375个）：先过滤技术性概念（含"涨停""次新""破净""连板"等关键词），再取近20日涨幅 **Top 50** 扫描

### Step 3: 板块深度评分（四维100分制）

| 维度 | 权重 | 说明 |
|------|------|------|
| 动量(30) | 30分 | 20日涨幅排名、5日加速、相对大盘超额收益 |
| 资金(30) | 30分 | 近5日/20日净流入金额、净流入占比 |
| 情绪(25) | 25分 | 板块内涨停股数量、连板高度、涨停股占比 |
| 结构(15) | 15分 | 板块指数是否在20日均线上方、量价配合 |

### Step 4: 主线确认

- 综合得分 ≥ 75 分 → **主线候选**
- 近 10 个交易日在主线候选中出现 ≥ 5 次 → **确认主线**
- 与历史数据对比，标记**新主线** / **老主线持续** / **老主线退潮**

### Step 5: 龙头挖掘

对每个确认主线，输出：
- **首个放量涨停**（若存在）：板块启动日最早放量涨停的个股
- **龙头候选 Top 5**：按近20日涨幅 + 成交额放大 + 市值排序，每只附带标签

### Step 6: 主线见顶预警

对确认主线检测以下预警信号：
- 板块内连板股数量连续 2 日下降
- 龙头股出现放量长上影/炸板
- 板块 20 日涨幅 > 40% 且 5 日涨幅 < 0
- 资金从净流入转为净流出连续 3 日
- 出现"高标接力失败"（最高板炸板后无新高板）

## 输出报告

```markdown
# A股板块主线追踪报告

## 一、大盘环境
| 日期 | 指南针活跃市值 | 日涨幅 | 多头状态 | 持续天数 | 沪深300 |

## 二、主线板块排名
| 排名 | 板块 | 类型 | 20日涨幅 | 资金净流入 | 涨停股 | 得分 | 状态 |

## 三、主线深度分析
### 🥇 主线1：XXX
- 龙头候选：A股(首板/连板X天/市值XX亿)、B股...
- 首个放量涨停：XXX（若存在）
- 见顶预警：⚠️ 连板股数量下降 / 🚨 资金连续净流出

## 四、主线历史持续性
| 板块 | 首次出现 | 持续天数 | 当前状态 |
```

## 使用方式

```bash
# 基本用法（用户提供指南针活跃市值 CSV）
python scripts/sector_leader_tracker.py --compass-csv ./data/compass_20260424.csv --output-dir ./reports

# 指定日期运行（复盘历史）
python scripts/sector_leader_tracker.py --compass-csv ./data/compass.csv --date 2026-04-24 --output-dir ./reports

# 查看数据库中的历史记录
python scripts/sector_leader_tracker.py --db ./data/sector_leader.db --query-history 30
```

## 故障处理

| 场景 | 处理方式 |
|------|---------|
| 指南针 CSV 缺失或格式错误 | 报错退出，提示用户提供正确格式 CSV |
| 某板块数据获取失败 | 跳过该板块，报告中标注，不影响其他板块 |
| 同花顺接口被反爬 | 自动切换到东方财富备用接口 |
| 涨停池数据为空 | 当日无涨停，情绪维度给中性分 |

## 与红利 Skill 联动

```json
{
  "date": "2026-04-24",
  "regime": "bull",
  "bull_days": 5,
  "leading_sectors": ["半导体", "AI算力"],
  "defensive_weight_recommend": 0.3
}
```

红利 skill 可读取此文件，在多头区间建议降低防御仓位、增配主线。
