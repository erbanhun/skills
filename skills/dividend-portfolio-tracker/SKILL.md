# 红利组合持有逻辑周度追踪器（批量版）

## 触发条件

用户提到以下任一关键词或意图时，使用本 skill：
- 红利组合 / 股息组合 / 红利篮子 / 股票清单
- 批量评估 / 批量评分 / 列表式 / 多只红利股
- 投资组合分析 / 红利股对比 / 持有逻辑清单
- 对「红利持有逻辑.md」中提到的股票进行分析
- 生成红利股评分表 / 周度检查列表

## 核心工作流

### 1. 运行追踪脚本

脚本位置：`scripts/dividend_portfolio_tracker.py`

**默认组合（14只红利蓝筹）**：
| 类别 | 标的 |
|------|------|
| 金融 | 工商银行、建设银行、农业银行、中国银行、中国平安 |
| 能源/资源 | 中国石油、中国石化、中国海油、中国神华 |
| 公用事业 | 长江电力 |
| 电信/运营商 | 中国移动、中国电信 |
| 消费/医药 | 贵州茅台、白云山 |

执行命令：
```bash
python scripts/dividend_portfolio_tracker.py --output-dir ./reports --json
```

参数说明：
- `--output-dir`: 报告输出目录
- `--json`: 同时输出原始评分 JSON
- `--codes`: 自定义股票代码（如 `--codes 600519 601398 601088`）
- `--names`: 自定义名称（与 `--codes` 一一对应，如 `--names 茅台 工行 神华`）

### 2. 读取并解读报告

脚本生成：
- `dividend_portfolio_report_YYYYMMDD.md` —— 列表式汇总报告
- `dividend_portfolio_data_YYYYMMDD.json` —— 每只股票的原始评分数据

报告包含四大部分：
1. **组合总览**：股票总数、成功评分数、平均得分
2. **评分汇总表**：按总得分降序排列的大表格，包含股息率、净利润增长、现金流、资产负债率、分红率、PE分位、总得分、评级
3. **分项得分明细**：股息率/健康度/估值/技术面四维拆解
4. **按类别分组**：对照《红利持有逻辑》的类别配置进行自检

### 3. 评分框架（与茅台skill完全一致）

| 维度 | 权重 | 说明 |
|------|------|------|
| 股息率定价 | 25分 | TTM股息率分档评分 |
| 分红健康度 | 40分 | 净利润增长、现金流/净利润、资产负债率、分红率 |
| 估值安全垫 | 20分 | PE历史分位 + 价格历史分位（5年） |
| 趋势技术面 | 15分 | 52周高点比、20周均线趋势 |

**评级标准**：≥80🟢 / 60-79🟡 / 40-59🟠 / <40🔴

### 4. 数据口径

- 行情：新浪财经前复权日线（5年回溯）
- 财务：巨潮资讯季度财务指标
- 分红：巨潮资讯历史分红明细
- 股息率：最近12个月现金分红 / 最新收盘价
- 分红率：年度每股分红 / 年度每股收益（取最近年报）

## 故障处理

| 场景 | 处理方式 |
|------|---------|
| 某只股票数据获取失败 | 跳过该标的，在报告中标注错误原因，不影响其他股票 |
| 组合太大运行慢 | 14只标的约需 2-5 分钟（取决于网络），可用 `--codes` 缩小范围 |
| akshare 未安装 | `pip install akshare pandas numpy` |
| 想增加/替换股票 | 使用 `--codes` 和 `--names` 参数，或修改脚本顶部的 `DEFAULT_PORTFOLIO` |

## 扩展建议

1. **每周一运行**：用任务计划程序定时生成报告，保存历史 JSON 做趋势对比
2. **再平衡参考**：报告中的「按类别分组」章节可直接作为年度再平衡的输入
3. **与茅台skill联动**：先用本skill扫描全组合，再用茅台skill深入分析重点标的
